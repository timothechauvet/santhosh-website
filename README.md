# Santhosh D Civil Engineering Homepage

A custom designed static website scaffolding generated by Stitch AI, refactored into a statically-rendered SvelteKit site automatically. Ready for deployment to GitHub Pages.

## Project Architecture

This project was bootstrapped with `npx sv create` and utilizes the following components:
- **Framework**: [SvelteKit](https://kit.svelte.dev/)
- **Styling**: TailwindCSS via CDN plugin with preset colors explicitly overridden in `src/app.html` header.
- **Components**: Consolidated Stitch HTML output refactored natively into `src/routes/+page.svelte` encapsulating interactivity.
- **SSG Engine**: Uses `@sveltejs/adapter-static` for simple static HTML generation configured dynamically for local deployment vs. sub-folder GitHub environments.

## Local Execution

To run locally and verify changes:
1. Ensure Node.js v20+ is installed.
2. Run `npm install` inside the project directory.
3. Run `npm run dev` to launch the local Vite development server.

## CI/CD Pipeline
Continuous integration is managed by GitHub Actions (`.github/workflows/deploy.yml`). Commits to `main` build via Node 20 and upload the custom `build/` directory over to GitHub Pages infrastructure.
